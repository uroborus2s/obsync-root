# 简介

`思齐（Stratix）` 是一个轻量、现代且功能强大的 Node.js 应用框架，它基于高性能的 [Fastify](https://www.fastify.io/) 构建，并从 [NestJS](https://nestjs.com/) 等优秀框架中汲取灵感，旨在提供一流的开发体验。

我们的设计哲学是 **“约定优于配置”** 与 **“强大的灵活性”** 相结合。通过提供合理的默认设置和自动化的工具，您可以快速启动项目；同时，通过模块化的插件架构和可深度定制的配置，`思齐（Stratix）` 也能满足复杂企业级应用的需求。

## 核心特性

- 🚀 **高性能**: 基于 Fastify，拥有业界领先的性能，是构建高并发 API 的理想选择。

- 💉 **强大的依赖注入**: 内置 `awilix` 容器，支持构造函数注入、生命周期管理（Singleton, Scoped, Transient），轻松实现控制反转 (IoC)。

- 🎨 **装饰器驱动**: 提供 `@Controller`, `@Get`, `@Post`, `@Executor` 等装饰器，让您以声明式、优雅的方式编写代码。

- 🔌 **插件化架构**: 框架的核心功能和业务模块都可以作为插件进行组织。每个插件都拥有独立的 DI 容器和生命周期，实现了真正的高内聚、低耦合。

- ⚙️ **双层生命周期管理**: 同时支持应用级的全局生命周期和插件内部的服务级生命周期，让资源管理和初始化逻辑清晰可控。

- 🔒 **企业级安全性**: 内置强大的 CLI 工具，支持对敏感配置进行加密、解密和密钥管理，保障应用安全。

- 🛠️ **完善的工具链**: 提供自动化的模块发现、路由注册和环境配置管理，让您专注于业务逻辑本身。

## 适用场景

- 构建高性能的 RESTful API 和 GraphQL API。
- 开发微服务架构中的后端服务。
- 编写复杂的后端任务和工作流（`@Executor`）。
- 作为企业级 Node.js 应用的统一开发框架。
