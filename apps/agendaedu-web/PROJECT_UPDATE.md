# AgendaEdu Web 项目更新

## 概述
已成功为 `@wps/agendaedu-web` 项目新增了左侧导航栏功能模块，并实现了美观的仪表板页面。

## 新增功能模块

### 1. 主要功能
- **仪表板** (`/dashboard`) - 包含统计卡片、图表和快捷操作的综合管理仪表板
- **数据查询** (`/data-query`) - 多维度查询学生出勤数据功能
- **统计分析** (`/analytics`) - 出勤率趋势分析和各学院对比分析
- **基础管理** (`/basic-management`) - 系统基础配置和管理功能

### 2. 系统管理  
- **任务管理** (现有功能扩展)
  - 任务列表 (`/tasks`)
  - 同步管理 (`/tasks/sync`)
  - 任务设置 (`/tasks/settings`)

- **课程管理** (新增)
  - 课程列表 (`/courses`)
  - 课表管理 (`/courses/schedule`)
  - 课程设置 (`/courses/settings`)

- **学生管理** (`/students`) - 学生信息管理功能
- **教师管理** (`/teachers`) - 教师信息管理功能

## 技术实现

### UI框架
- 使用 shadcn/ui 组件库
- TailwindCSS 样式系统
- Lucide React 和 Tabler Icons 图标

### 路由系统
- TanStack Router 文件系统路由
- 自动类型生成和类型安全
- 路由级别的代码分割

### 数据管理
- TanStack Query 数据获取和缓存
- TypeScript 严格类型检查
- 模拟数据展示功能效果

### 响应式设计
- 移动端和桌面端适配
- 优雅的卡片布局
- 交互式图表和进度条
