## 4. 风险与应对

| 风险 | 影响 | 触发信号 | 应对策略 |
|---|---|---|---|
| 根脚本继续失真 | 验证结果误导 | `pnpm build` / `pnpm test` 与包级结果冲突 | 先修根入口，再谈 CI |
| 核心包回归扩大 | 阻塞发布 | `@stratix/core` build/test 继续失败 | 作为优先级最高的包级 bug |
| 发布口径继续分裂 | 无法交付 | manifest/tag/registry 持续不一致 | 单独作为 `CR-001` 推进 |
