# 现状调研概览

本目录收纳历史项目纳管输入、现状核查、差距分析和问题空间说明，回答“现在真实是什么样、还缺什么证据”。

## 适用对象

- 负责项目纳管和阶段判断的协作者
- 需要核对当前真实状态的维护者

## 维护约束

- 当前状态结论优先写回 [当前真实状态分析](./current-state-analysis.md)。
- 调研页只记录经过核对的事实、输入背景和分析，不把猜测写成结论。
- 重要复评报告需要同步到根级 `docs/index.md` 导航，并在 `.factory/memory/current-state.md` 记录证据摘要。
