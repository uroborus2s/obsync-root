# 项目概览

灵枢枢机（Stratix）是一个基于 Node.js / TypeScript 的 `pnpm` + `turbo` monorepo，当前仓库同时维护框架核心、CLI、生态插件、文档体系，以及可独立运行的前端预览样例。

## 仓库包含什么

- `packages/`：公共框架包与生态插件
- `examples/web-admin-preview/`：幻廊之镜前端脚手架可运行样例
- `docs/`：正式人类文档源
- `.factory/`：AI 记忆、项目状态和工作项控制层

## 文档怎么组织

- `01-getting-started/`：项目概览、快速开始和文档地图
- `02-user-guide/`：面向使用者与接手者的说明
- `03-developer-guide/`：面向开发者的应用后端开发、脚手架开发和插件开发指南
- `04-project-development/`：框架自身研发过程文档
